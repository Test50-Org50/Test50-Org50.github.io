# A. Length

## Problem Setup

- login to [CS50 IDE](https://cs50.dev)
- write `cd` in the *terminal* to go to the home directory
- write `mkdir length` to create a folder called `length`
- write `cd length` to go to the `length` folder
- write `code length.cpp` to create a file called `length.cpp` and open it in the editor

## Problem

Given a string \\(S\\) consisting of lowercase English letters and uppercase, print the length of the string.

## Input

The first line contains a number \\(T\\) (\\(1 \\leq \|T\| \\leq 100\\)) number of test cases.

The second line containing the string \\(S\\) (\\(1 \\leq \|S\| \\leq 1000\\)).

## Output

For each test case, print the length of string.

## Examples

**Example 1:**

**Input:**

```text
2
NewComers
PortSaid
```

**Output:**

```text
9
8
```
